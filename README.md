# 🎮 Pokémon Backend API

Sistema backend inspirado no universo Pokémon focado em:
- exploração
- encontro de Pokémon
- captura
- Pokédex do jogador

O projeto foi desenvolvido utilizando arquitetura organizada e escalável, simulando uma jornada Pokémon na região de Kanto.

---

# 📌 Funcionalidades

## ✅ Sistema de Exploração
O jogador pode:
- iniciar jornada
- explorar a região de Kanto
- encontrar Pokémons aleatórios

Os encontros funcionam através de probabilidade baseada em raridade.

## ✅ Sistema de Captura

Ao encontrar um Pokémon:
- o jogador pode tentar capturá-lo
- a captura funciona através de chance/probabilidade
- Pokémons mais raros possuem menor taxa de captura

---

## ✅ Pokédex

O sistema possui:
- armazenamento de Pokémons capturados
- listagem da Pokédex
- detalhes dos Pokémons
- histórico de captura
